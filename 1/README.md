# First Setup

### get the list of installed libs.

```shell
>>> help('modules')
```

### get the system-dependent version information.

```shell
>>> import os
>>> os.uname()
```

### Get the current directory.

```shell
>>> os.chdir(path)
```

### Change current directory.

```shell
>>> os.getcwd()
```

### Create a new directory.

```shell
>>> os.mkdir(path)
```
